# 刺客信条·传国玉玺（原创剧情）

一部以「刺客信条」世界观为背景的**原创**剧情创作：圣殿骑士王莽借伊述遗物之力，将意识逆溯回西汉末年，企图以传国玉玺统御历史；而历史，最终被一个老妇人的盛怒与一个布衣的失手，拨回了正轨。

> 沿用刺客兄弟会 / 圣殿骑士团、伊述遗物等设定，人物、情节与台词均为原创，与育碧官方作品无关。

## 已完成

- **第一部·玉玺（公元 9 年）** — 一明一暗双线分镜：明线王莽登基受贺，暗线王舜携杜吴入长乐宫索玺，杜吴阴差阳错崩坏玉玺、下狱。
- **第二部·刺杀刘秀** — 以「刘秀的悬赏令」引出：明线举国通缉指错方向，暗线王莽凭未来记忆锁定真刘秀、密遣杜吴暗杀。
